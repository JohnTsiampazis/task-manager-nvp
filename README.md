# Task-Manager-MVP

A minimal, fully functional Todo app build with React.

-Add Task: Adds new tasks
-Edit Task: Inline editing of existing tasks
-Delete Task: Remove tasks
-Complete Toggle: Marks tasks as completed on click
-Filters: View ALL/ACTIVE/COMPLETED tasks
-LocalStorage: Tasks persists even after refreshing the page
